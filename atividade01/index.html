Table
Tag utilizada para criar uma tabela.
Tag Th
A tag th é utilizada para definir uma célula de cabeçalho.
Tag Tr
Essa tag é utilizada para definir uma linha na tabela
Tag Td
Td serve para criar uma nova células da tabela
Thead
Essa tag tem uma funcionalida de agrupar todo o conteúdo que formara o cabeçalho da tabela.
Tbody
Essa tag tem a funcionalidade de agrupar todo a estuturura de corpo da tabela.
Align
Esse atribulto tem como fucionalidade de alinhar os elementos.
Border
Esse atribulto tem como funcionalidade por uma borda em elementos, se ele for colocado na tag table ele ira por borda em todos os elementos da table
Caption
Esse atribulto serve para definir um titulo para a tabela.
Rowspan
Esse atribulto serve para expandir as células fazendo com que uma única célula ocupe mais de uma linha. (Trabalhando na Vertical)
Colspan
Esse atribulto permite que uma célula ocupe várias colunas, permite mesclar colunas. (Trabalha na Horizontal)
Width
Esse atribulto serve para definir a largura de um elemento.
